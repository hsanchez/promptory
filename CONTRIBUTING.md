
# Contributing to Promptory

## Getting Started

1. Fork the repository and clone your fork.
2. Install dependencies: `uv sync`
3. Run the test suite to confirm everything works: `uv run pytest`

## Reporting Issues

- Search existing issues before opening a new one.
- Use the provided issue templates for bug reports and feature requests.

## Submitting Pull Requests

1. Create a branch from `main`: `git checkout -b your-feature`
2. Make your changes and add tests where applicable.
3. Ensure all tests pass: `uv run pytest`
4. Open a PR against `main` and fill out the PR template.

## Coding Standards

- Follow existing code style (PEP 8, type hints where present).
- Keep changes focused — one concern per PR.
- Do not add dependencies without discussion in an issue first.

## Questions

Open a GitHub Discussion or issue rather than emailing maintainers directly.
